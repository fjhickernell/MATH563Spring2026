# Hickernell Test Archive

Shared archive of past tests and exams for statistics courses.

Structure:
- One folder per course (476, 563, 565)
- Flat within each folder (year/term encoded in filename if present)
- Files may be duplicated across courses if a test applied to both

This repo is intended to be used as a submodule at:
assets/tests/archive/

in course Quarto websites.
